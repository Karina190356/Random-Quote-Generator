## Описание программы

«Random Quote Generator» — это кроссплатформенное десктопное приложение с графическим интерфейсом (*GUI*) для генерации случайных вдохновляющих цитат. Приложение позволяет просматривать предопределенные цитаты по темам и авторам, а также добавлять собственные высказывания в базу данных. История сгенерированных цитат сохраняется между запусками программы.
> **Примечание:** Библиотека **ttkthemes** используется для красивого оформления интерфейса (темы), но **не является обязательной**. Если она не установлена или недоступна для импорта (*ImportError*), приложение запустится со стандартным стилем *Tkinter*.
> **Важное замечание:** Приложение выполняет строгую **валидацию данных** при добавлении новой цитаты (поля 'Текст' и 'Автор' обязательны).

## Примеры использования и тесты

Ниже приведены примеры сценариев использования приложения для проверки его функциональности.



### Сценарий 1: Генерация случайной цитаты и проверка истории

1. Запустите приложение (`python main.py`).
2. Нажмите кнопку «Сгенерировать цитату».
3. **Ожидаемый результат:** В верхней части окна появится текст новой случайной цитаты с указанием автора и темы.
4. Нажмите кнопку еще раз несколько раз.
5. **Ожидаемый результат:** В поле «История» появится список всех ранее сгенерированных вами цитат в порядке их появления. История сохраняется после закрытия программы и загружается при следующем запуске.
6. Закройте программу и запустите снова (`python main.py`).
7. **Ожидаемый результат:** История из предыдущего сеанса должна быть загружена и отображена в списке «История».



### Сценарий 2: Добавление новой собственной цитаты через форму

1. Запустите приложение (`python main.py`).
2. Нажмите кнопку «Добавить свою цитату» -> «Добавить цитату».
3. В открывшемся диалоговом окне заполните поля:
   - Текст цитаты: `Тестирование — это не поиск ошибок, а сбор информации.` *(обязательное поле)*
   - Автор: `[Ваше Имя]` *(обязательное поле)*
   - Тема: `Программирование`
4. Нажмите кнопку «Добавить».
5. **Ожидаемый результат:** Появится сообщение «Цитата успешно добавлена в базу данных!». Диалог закроется.
6. Нажмите «Сгенерировать цитату» несколько раз до тех пор, пока ваша новая цитата не появится случайно среди других.
7. **Ожидаемый результат:** Ваша новая цитата успешно интегрирована в общую базу данных и может быть выбрана случайным образом генератором.



### Сценарий 3: Фильтрация истории по автору и теме

1. Запустите приложение (`python main.py`).
2. Сгенерируйте несколько разных цитат от разных авторов (например, Конфуций и Стив Джобс).
3. В поле фильтра «Автор» выберите из выпадающего списка «Конфуций».
4. Нажмите кнопку «Фильтр по автору».
5. **Ожидаемый результат:** В списке истории останутся только те строки, где автором является Конфуций.
6. Нажмите кнопку «Сбросить фильтр».
7. В поле фильтра «Тема» выберите из выпадающего списка «Мотивация».
8. Нажмите кнопку «Фильтр по теме».
9. **Ожидаемый результат:** В списке истории останутся только те строки, темой которых является «Мотивация».



## Запуск тестов

Проект содержит модульные тесты для проверки логики работы с данными (генерация случайного числа, фильтрация).
Для их запуска убедитесь что виртуальное окружение активировано и выполните команду из корня проекта:
bashpython -m unittest discover -s tests/


Вы должны увидеть вывод вида `OK`, что означает успешное прохождение всех тестов.
> *Тесты используют отдельную папку `tests/data_test`, чтобы не изменять основной файл данных приложения.*



## Формат файла данных (`data/quotes.json`)
Приложение хранит данные в формате *JSON*. Это массив объектов-цитат. Каждый объект должен иметь следующую структуру:
json[   {      "text": "<строка с текстом>",      "author": "<строка с автором>",      "topic": "<строка с темой>"   },   ...]

Копировать
*   **text**: Текст самой цитаты (*String*, не пустое).
*   **author**: Автор высказывания (*String*, не пустое).
*   **topic**: Категория или тема (*String*, может быть пустой).
> *Поля чувствительны к регистру.*
